# FlappyBird-ProjectLearn-1

A Unity 2022 LTS learning project built around a Flappy Bird style game.

## Project Info

- Engine: Unity `2022.3.62f3 LTS`
- Render Pipeline: `URP 2D`
- Main scene: `Assets/Scenes/SampleScene.unity`

## Goal

This project is used to learn:

- Unity 2D workflow
- Clean gameplay scripting
- OOP, SOLID, and low-coupling architecture
- Reusable movement and environment loop systems

## Folder Layout

- `Assets/Project/Assets`: sprites, audio, UI assets
- `Assets/Project/Prefab`: prefabs
- `Assets/Project/Script`: gameplay and environment scripts
- `Assets/Scenes`: playable scenes
- `ProjectSettings`: Unity project configuration

## Open The Project

1. Open Unity Hub.
2. Add this folder as an existing project.
3. Use Unity version `2022.3.62f3`.
4. Open `Assets/Scenes/SampleScene.unity`.
5. Press Play in the editor.

## Notes

- `Library`, `Temp`, and other generated folders are excluded from Git.
- The repository tracks only source assets and Unity settings required to rebuild the project locally.
